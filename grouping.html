Grouping Selector

The CSS grouping selector is used to select multiple elements and style them together. This reduces the code and extra effort to declare common styles for each element. To group selectors, each selector is separated by a space.In this chapter, you will learn about the Group Selector in CSS. As you have used the different CSS selectors that only target either a class or an ID or a single element for applying the styles. Nevertheless, CSS also allows making it incredibly easy for targeting more than one element at a time to allow for specific CSS properties and rules.
